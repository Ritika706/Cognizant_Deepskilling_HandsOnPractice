
Linear Search:
- Checks each element one by one.
- Works on sorted and unsorted data.

Binary Search:
- Works only on sorted data.
- Repeatedly divides the search space into halves.

Time Complexity:

Linear Search
Best Case  : O(1)
Worst Case : O(n)

Binary Search
Best Case  : O(1)
Worst Case : O(log n)

When to Use:

Linear Search:
- Small datasets
- Unsorted data

Binary Search:
- Large datasets
- Sorted data
- Faster searching
